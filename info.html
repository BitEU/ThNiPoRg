What is Stunts?

Stunts is a racing car game which was made in the early nineties. There is a big choice of cars and tracks. One of the best parts of the game is that you can create your own tracks with a built in editor. 

There are 3 main versions of the game: 

Version 1 was made for the Amiga and is called "Stunts". This version has the best graphics and better sound. So if you still have a Amiga in your garage you might want to check this game out. 

Version 2 is for the PC. This DOS version is not as good as its Amiga counterpart, but it still runs on todays machines, also Windows 95. This Site is mainly for the PC version of the Game. 

Version 3 is called "4 D Sports Driving". The game is part of the "4 D Sports" Series. Why 4 D? Thats because it has a 4th dimension in the Game, which is TIME. At any given moment you can rewind and keep on playing from that point. This is another reason why this game is so special.
